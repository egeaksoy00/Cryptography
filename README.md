# Cryptography

A collection of small, educational cryptography projects written in Java.  
This repository serves as a hands-on playground for learning fundamental concepts of classical encryption, brute-force attacks, key spaces, and simple cryptanalysis techniques.

Each mini-project focuses on a different cipher or cryptographic idea, helping build strong foundations for more advanced topics like hashing, symmetric/asymmetric encryption, and modern security protocols.

The goal of this repository is to:
